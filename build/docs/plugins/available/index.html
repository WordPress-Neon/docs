Plugins within the WordPress Neon framework and snippets of code that can be enabled and disabled depending on your requirements. They are intended to speed up development by giving you an implementation of the code modern themes include everytime, such as an options' menu with the theme logo, SVG support and so on.

Enabling WPN Plugins

Plugins are enabled by referencing the fully qualified class name in the applications' config file, in the plugins block. Plugins are registered during the applications' initialisation method called in the AppServiceProvider.
